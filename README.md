# Student Daily Life Manager (WinForms)

##  Project Description
Student Daily Life Manager is a Windows Forms application developed using C# and SQL Server.
This system helps students manage their daily academic and personal activities efficiently.

##  Main Features
- User Registration, Login & Logout
- Course Management (Add, Edit, Delete, View)
- Class Routine Management
- Assignment Management with Due Tracking
- Exam Schedule & Upcoming Exam View
- Expense Tracking
- Monthly Budget Management
- Admin User Control (Block / Unblock / Monitor)
